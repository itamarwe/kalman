Device orientation smoothing

This example smooths alpha, beta, and gamma from the browser deviceorientation event. It models the current orientation angles directly, so the state vector is [alpha, beta, gamma].

Increase Q when the filter feels too slow. Increase R when a sensor channel is too noisy. In this example alpha is treated as noisier than beta and gamma. 

var x_0 = $V([alpha, beta, gamma]); // current angle estimate
var P_0 = diagonal([1, 1, 1]);      // initial uncertainty
var F_k = identity3();              // next angle starts near previous angle
var Q_k = diagonal([0.03, 0.01, 0.01]); // true motion between readings

var z_k = $V([alpha, beta, gamma]); // current sensor reading
var H_k = identity3();              // measured angles map directly to state
var R_k = diagonal([0.12, 0.04, 0.04]); // sensor noise per channel
